# Bookseller
Next.js used book platform I made with TailwindCSS

## Getting Started
First, run the development server:
```bash
npm run dev
# or
yarn dev
```

## Built With
- [Next.js](https://nextjs.org/)
- [TailwindCSS](https://tailwindcss.com/)
